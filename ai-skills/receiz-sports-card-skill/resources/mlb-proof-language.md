# MLB Proof Language

Use exact, proof-bound sports language.

## Safe Language

- "MLB-themed Receiz sports card"
- "Player identity in the card proof"
- "Pitch witness proof"
- "Live event append"
- "Pack opening proof"
- "Card memory"
- "Market projection"

## Unsafe Language

- Official league endorsement unless the proof or user-provided source establishes it.
- Player statistic claims without a verified source.
- Team or roster claims inferred from memory.
- Value claims without a market or settlement proof boundary.

## Explanation Pattern

"This card carries player identity, media, and card memory as proof object state. Live events append to that memory when verified."
