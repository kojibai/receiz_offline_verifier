# Sports Card Object Model

A Receiz sports card is a proof object with player identity, card identity, media, rarity, provenance, card memory, ownership, and public proof.

## Fields To Inspect

- Card proof identity.
- Player name and team fields present in the proof.
- Media and visual identity.
- Rarity or edition field, when present.
- Pack origin or issuance route.
- Event memory and witness IDs.
- Ownership or custody state.
- Market projection and value display.

## Stronger Truth

Card proof and verified event appends outrank market display, game UI, social copy, and database state.

## Projection Rule

If a card market or game surface disagrees with card proof, report the card proof first and mark the projection as inconsistent.
