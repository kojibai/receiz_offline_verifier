# Game Surface Patterns

Game surfaces can project sports proof into play, but they do not outrank card memory.

## Required Surfaces

- Card proof identity.
- Event memory.
- Game action or score.
- Ownership or custody when relevant.
- Market projection when value is displayed.

## First Paint

Known card proof and known game proof should paint immediately. Background sync may append verified events.

## Copy

Use direct labels:

- Card proof
- Live event memory
- Pitch witness
- Pack proof
- Ownership state
- Market projection
