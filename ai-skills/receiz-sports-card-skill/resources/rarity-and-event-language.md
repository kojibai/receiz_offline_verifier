# Rarity And Event Language

## Rarity

Use rarity language only when supported by proof, event append, release law, or manifest field.

Allowed:

- "Rarity is proven by the card manifest."
- "Rarity is not present in the available proof."
- "This event append affects card memory, not base rarity."

Forbidden:

- "Rare" because the card looks unusual.
- "Limited" without edition proof.
- "Legendary" without release or manifest evidence.

## Event Language

Use append language:

- "A live event was appended to card memory."
- "The pitch witness is verified."
- "The event is not proven by the provided data."
