## Release Decision

Ship `v88.0.0` only when:

- version surfaces are coherent
- release lock passes
- focused Sports and DB hardening contracts pass
- full promotion gates pass or exact failures are documented as release blockers
- visual Sports evidence confirms score plate, pitch-by-pitch, last play, and live player cards after game switching
- production Supabase migration status is confirmed for the exact production project
- no stronger proof/local/artifact truth is replaced by weaker server/session/UI state
