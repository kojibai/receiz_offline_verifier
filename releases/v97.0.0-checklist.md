## Release Blockers

- missing v97.0.0 release docs
- missing v97.0.0 regression lessons, performance findings, invariant register, or commit-history boundary
- version surface mismatch
- SDK package version drift from app version
- SDK runtime constant drift from package version
- app-state SDK helpers missing from package evidence
- public app-state projection routes missing or not URL-normalized
- OIDC consent redirect/readability regression
- public proof projection returning weaker verification URLs or losing owner-scoped behavior
- admin performance analytics measuring internal/admin surfaces instead of user-visible app experience
- SDK clients, OpenAPI docs, public app-state projections, server, or DB surfaces implying authority above proof objects, identity primitives, durable local truth, verified appends, or Kai/Klok state
