## Frozen Invariants

- Receiz is a proof-native artifact system.
- Implemented primitives are named directly.
- A settled visible shell is primary reality.
- A settled visible shell is immutable unless the user explicitly interacts.
- Stronger truth may be decorated, synced, indexed, hydrated, displayed, or appended to by weaker truth.
- Weaker truth never replaces stronger truth.
- Weaker truth never renames stronger truth into a weaker category.
- Sealed artifact truth outranks server, DB, session, UI cache, and generic app convention.
- Durable local truth is not cache.
- Offline proof is not fallback mode.
- Public proof surfaces are not generic share pages.
- Ownership surfaces are not dashboard state.
- Settlement primitives are not balance displays.
- Deterministic first paint is not loading state.
- First paint must not serialize non-critical data to prove a visible shell that already has stronger truth.
- First paint must not paint empty/null when minimum visible truth is already available.
- SEO, analytics, metadata, personalization, DB refresh, session check, hydration, lazy import, dynamic JSON serialization, global sync, server reconciliation, tracking, marketing, and external framework convention never block or downgrade deterministic first paint.
- Background work may warm, append, hydrate, index, or refine only when it does not disturb the settled surface.
- If the app already knows stronger local/artifact truth, it reuses that truth immediately.
- Any change that makes a previously settled deterministic surface feel like it loads again is a regression.
- Agents run the primitive decision procedure before modifying primitive surfaces.
- Vague global uncertainty is forbidden. Exact uncertainty boundaries are required.
- External systems may inform how Receiz is built. They may never define what Receiz is.
- Generic convention is not neutral authority. Hedge weak inherited priors before strong implemented primitives.
- The user must not have to restate this hierarchy.

- Production Supabase targets must have every Sports, Signal Run vault, profile/public asset, original engagement, and refunded tournament entry migration applied.
