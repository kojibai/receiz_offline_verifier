## Release Blockers

- missing v96.1.0 release docs
- missing v96.1.0 regression lessons, performance findings, invariant register, or commit-history boundary
- version surface mismatch
- SDK package version drift from app version
- SDK runtime constant drift from package version
- Twin or World SDK clients missing from package evidence
- Connect Twin scopes missing from developer discovery
- OpenAPI/status catalog drift from implemented developer routes
- SDK clients, OpenAPI docs, or status docs implying authority above proof objects, identity primitives, durable local truth, verified appends, or Kai/Klok state
