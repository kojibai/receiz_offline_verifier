## Release Decision

Ship `v89.0.0` only when:

- version surfaces are coherent
- release lock passes
- focused anchor/profile/PWA contracts pass
- full promotion gates pass or exact failures are documented as release blockers
- production deployment/cache state is confirmed after promotion
- BJ Klock end-to-end production post evidence is captured from a BJ Klock authenticated session
- no stronger proof/local/artifact truth is replaced by weaker server/session/outbox/UI state
