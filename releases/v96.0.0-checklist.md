## Release Blockers

- missing v96.0.0 release docs
- missing v96.0.0 regression lessons, performance findings, invariant register, or commit-history boundary
- version surface mismatch
- SDK package version drift from app version
- SDK CLI, local conformance, pack dry-run, or package contents missing from broader release evidence
- SDK local conformance using network or database authority
- copy-paste integration docs teaching rediscovery before admitted proof memory
- public report or developer docs carrying weaker proof-object language
