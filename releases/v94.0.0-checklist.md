```bash

```

The published package must be `@receiz/sdk@94.0.0`.

## Release Blockers

- missing v94.0.0 release docs
- missing v94.0.0 performance findings or invariant register
- version surface mismatch
- SDK package version drift from app version
- SDK schema/projection/proof memory tests missing or failing
- SDK publication attempted before build/test/package gates pass
- SDK helper treated as proof authority above sealed artifacts or verified appends
