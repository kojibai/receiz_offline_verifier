## Release Blockers

- missing v97.5.0 release docs
- missing v97.5.0 regression lessons, performance findings, invariant register, or commit-history boundary
- version surface mismatch
- SDK package version drift from app version
- MCP package version drift from app version
- MCP initialize version drift
- delegated agent tokens implying identity proof authority
- MCP tools implying authority above proof objects
- SDK or MCP helpers replacing proof objects, identity primitives, ownership appends, settlement rows, durable proof memory, public projections, verified appends, or Kai/Klok state
