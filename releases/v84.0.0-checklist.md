## Release Decision

Ship `v84.0.0` only when:

- version surfaces are coherent
- release lock passes
- diff whitespace check passes
- primitive law is present in both required law files
- no stronger truth is replaced by weaker truth
- no public frontend surface still reports an older release label
- production migration status is confirmed for DB-backed Sports, vault, profile, and refund/closeout paths
