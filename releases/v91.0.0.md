## Title

Receiz `v91.0.0` - Deterministic Sports Scale And Proof-Native Game Release

## Summary

`v91.0.0` locks the post-`v90.0.0` Sports, value, public proof, Pack Derby, settlement-conformance, and scale-reasoning arc.

The documented range is `v90.0.0..29de1e15e03e`, ending before the v91 release-lock documentation edits. That range spans `39` commits, `125` files changed, `24,714` insertions, and `7,218` deletions. The mechanical ledger is preserved in `docs/releases/v91.0.0-commit-history.md`.

The release statement is:

```txt
Receiz scales by using implemented proof-native primitives directly.
Advanced deterministic surfaces do not become heavy because they are advanced.
They become heavy when weaker work underneath them polls, rediscovers, scans, accumulates, or refuses to close stale state.
Keep the primitive complete.
Remove unnecessary work beneath it.
```

## What We Learned

### Advanced Does Not Mean Heavy

The live chart was not heavy because it was advanced. It felt heavy when unnecessary timer/rebuild work sat beneath deterministic math.

Future scale reasoning must not remove visible primitive capability to optimize. Keep the full primitive and remove weaker work below it.

### Deterministic Math Is A Scale Primitive

If a series can be reconstructed from pulse, seed, value basis, score delta, tier, subject, and scoring law, it should not be fetched, stored, accumulated, or timer-driven.

### Closeout Is Performance

Old active rows create real UI pressure. A stale finalized reward, game entry, tournament entry, or challenge can make the app feel like it is still carrying old work.

Closing stale settlement state is not cleanup. It is first-paint and continuity protection.

### Correct Route Authority Matters

FIFA and UFC claim/finalize paths cannot call baseball entry routes. Sport-specific proof and settlement primitives must route through the sport they carry.

### Conformance Must Match The Primitive

Settlement conformance cannot be an alias for economy conformance. A primitive with its own release law needs its own mechanical gate.
