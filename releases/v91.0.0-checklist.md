## Release Decision

Ship `v91.0.0` only when:

- version surfaces are coherent
- release lock passes
- focused Sports, value, settlement, and documentation checks pass
- full promotion gates pass or exact failures are documented as release blockers
- production deployment/cache state is confirmed after promotion
- live production DB and authenticated-session evidence boundaries are named exactly
- no stronger proof, local, artifact, ownership, settlement, identity, or public proof truth is replaced by weaker server/session/outbox/UI state
