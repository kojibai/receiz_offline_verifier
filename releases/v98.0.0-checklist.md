## Release-Lock Requirement

`pnpm release:lock 98.0.0` must require this checklist for `v98.0.0` and later release cuts. A release that omits the checklist is incomplete because the post-`v97.0.0` SDK/MCP, delegated-token, webhook, public proof graph, ledger/transaction, market/profile, sports performance, Pitch Command archive, performance findings, invariant register, and publication boundaries would not have an explicit gate list.
