## Actual Anchor Issue

The foreground anchor route was not the broken authority.

The root regression was introduced by commit `921a00395826f0a6336b6bcb83b1ba7e2ead007d` on June 8, 2026. That change generalized local signed-action/outbox behavior so foreground proof publication could be admitted to the generic outbox path.

That violated the truth hierarchy:

```txt
foreground proof publication route
sealed/deterministic anchor response
profile original publication projection
public profile showcase/global sync/market projection
generic local signed-action outbox
```

`/api/receiz/anchor` is the foreground proof publication route. It owns its own anchor response contract: readable JSON with `{ ok, anchorId, anchoredUrl }` on success, and readable JSON errors on failure. The generic local signed-action outbox is a weaker replay/sync/publication layer. It may not sign, replay, inspect, wrap, or consume the foreground anchor response path before the composer reads it.

The production symptom was:

```txt
The anchor route returned an unreadable response. Refresh and retry.
Could not anchor this memory payload.
```

The server route could still return readable JSON. The failure came from allowing the weaker outbox boundary to touch the foreground anchor response path.

The fix is commit `ad246053cfdf8510c57c01fc0c7901014270b3d0`, `Exclude anchor route from local signed outbox`.

Locked rule:

```txt
/api/receiz/anchor is excluded from local signed-action outbox admission.
No broad /api/receiz proof classifier may re-admit the foreground anchor route.
```

## Product Statement

`v89.0.0` exists because foreground proof publication must never be routed through weaker replay/sync mechanics.

Receiz may use servers, DB rows, snapshots, outboxes, service workers, and market append sources. Those are projection and sync mechanics beneath the primitive. They do not own the truth boundary for a sealed proof object, foreground anchor response, profile original, Sports Vault image, public proof surface, identity primitive, or settlement primitive.
