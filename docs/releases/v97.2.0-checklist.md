## Release Blockers

- missing v97.2.0 release docs
- missing v97.2.0 regression lessons, performance findings, invariant register, or commit-history boundary
- version surface mismatch
- SDK package version drift from app version
- SDK runtime constant drift from package version
- React subpath package/export drift
- enterprise SDK helpers missing from package evidence
- idempotency headers missing on write helpers
- offline proof queue replay duplicating verified additions
- SDK clients, React components, OpenAPI docs, public app-state projections, server, or DB surfaces implying authority above proof objects, identity primitives, durable local truth, verified appends, or Kai/Klok state
