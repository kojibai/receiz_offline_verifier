- Known verified truth moves first, and weaker stale echoes cannot take away account, profile, wallet, market, card, vault, or creator truth after proof acceptance.
- `/<username>` remains canonical profile truth, and `?manage=1` remains a permission overlay only.
- Identity Record, Identity Seal, Receiz Key, PBI, email, server session, and Receiz ID restore/project into the same signed-in product experience.
- Wallet, Settlement, Reserve, market shares, cards, auctions, packs, transfers, and battle stakes preserve funded causality and append-only value truth.
- Signal Cards are owned Receiz assets across game, world, vault, discovery, transfer, auction, battle, market, export, share, and conformance surfaces.
- Share-card and proof popovers stay bounded instead of becoming full-screen replacement surfaces.
- Signal Card discovery, card pages, battle queue, and card-stage controls fit mobile, preserve owner scope, and keep interactions settled.
- Vault Verify enters explicit opening state only after user click.
- Vault export and download are user-intent-only flows and do not use hidden iframes, synthetic blank anchors, panel-open timers, or mobile/PWA paths that navigate away.
- Transfer packages make ownership movement explicit and avoid ambiguous duplicate ownership state.
- Arena append proof manifests expose card battle/appended proof without mutating original card truth.
- Card flip gestures remain in a dedicated interaction path with contract coverage.
- Game-front copy and layout read as the official Signal Circuit surface, not a prototype.
- Creator OS and World agent loops produce durable artifacts, charge or hold wallet value explicitly, and keep first paint settled while private work enriches behind it.
- Public Proof, Signal Circuit conformance, Status, and Exchange make the public system inspectable without publishing private/admin/worker routes.
- Runtime and service-worker coherence prevent visible second loads, visible route takeovers, stale header state, and stale worker state.
- Release surfaces move together to `v80.0.0`.
