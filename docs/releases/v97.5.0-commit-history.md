# Receiz v97.5.0 Commit History

Release date: 2026-06-30

Baseline: `v97.0.0`

Commit count: local release-preparation working tree

This release collects the public app-state, `@receiz/sdk`, and `@receiz/mcp-server` work completed after `v97.0.0`.

## Release Delta

- Bumped root release identity to `v97.5.0`.
- Bumped `@receiz/sdk` to `97.5.0`.
- Bumped `RECEIZ_SDK_VERSION` to `"97.5.0"`.
- Bumped `@receiz/mcp-server` to `97.5.0`.
- Bumped MCP initialize `serverInfo.version` to `"97.5.0"`.
- Regenerated SDK `dist` output.
- Regenerated MCP `dist` output.
- Added durable public app-state and public-store projection rails for third-party apps.
- Added SDK support for tenant customer sessions, customer portal, merchant onboarding, one-click checkout, media, domains, events, jobs, webhooks, permissions, audit, risk, compliance, portability, search, notifications, release pinning, sandbox, React hooks, hosted UI helpers, and offline proof queues.
- Added the published MCP server package for agents building and verifying Receiz apps.
- Added Receiz-issued delegated agent tokens through `/api/oidc/agent-tokens` and `/developers/connect`, with one-time bearer output, token ID revocation, exact OIDC scopes, and `agent:*` audit labels.
- Added `receiz_mcp_login` so MCP hosts can discover the delegated agent-token setup path and local `RECEIZ_ACCESS_TOKEN` config without treating MCP as authority.
- Fixed MCP binary startup through npm `.bin` symlinks by comparing real paths.
- Added MCP tools for offline file verification, public asset lookup by URL/ID, pitch proof witness lookup, card history, transfer preview, transfer confirmation policy, pack-open preview, marketplace template generation, and resource templates.
- Added `/developers/mcp`.
- Linked MCP from `/developers`, `/developers/openapi`, and `/developers/connect`.
- Locked developer contracts around MCP package discovery, correct `npx -y @receiz/mcp-server` command, and authority boundary.
