## Deployment Order

1. Apply pending Supabase migrations.
2. Deploy `93.0.0` app code.
3. Check DB/API logs for missing RPC or schema-cache errors.
4. Reset `pg_stat_statements`.
5. Re-run Sports Economy, Events, History, card proof, Pitch Command, Play Command, Pack Derby, and physical activation smoke flows.
6. Capture or review visual evidence for any surface where the latest browser state contradicts a claimed pass.

## Release Blockers

- missing v93 release docs
- version surface mismatch
- missing Supabase RPC/function signature
- visible proof/history/value regression
- slow-query timeout recurrence on a patched hot path
- first-paint regression on Sports Economy or proof surfaces
- generated build/type failure from a clean release build
- production DB errors after migration/deploy
