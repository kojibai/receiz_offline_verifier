# Receiz v107.0.0

Release date: 2026-07-17

Receiz v107.0.0 ships unified developer operations across Receiz.com, `@receiz/sdk`, `@receiz/mcp-server`, `@receiz/ai-skills`, the frontend, service worker, emulator, generator, migration, registry, and release evidence.

The release exposes the proof-native identity, profile, media, portable continuity, generic bearer ownership, receipt, proof-head, and offline-command primitives already carried by Receiz.com. The SDK and MCP server project those primitives; they do not replace sealed artifact truth or create a competing authority.

The key remains the identity. Renaming a username appends to the original identity, and the prior username becomes available only after the new claim commits. Raw profile image files and existing media proof references use one atomic profile/media/head append. Generic bearer claims apply to every qualifying Receiz artifact and verify the enclosing artifact before reading embedded payloads.

Offline work is a signed proposal. `status: "queued"` always includes `globallyCommitted: false`; only a verified canonical receipt can establish a committed append. Known verified local truth paints immediately and sync discovers verified additions after the known head.

All coordinated packages and runtime version surfaces are exactly `107.0.0`, with registry digest `4d0caa6172a69c3bf5817c1c35db5630e555b5d6d824091d45a90fb426b86ef6`. The package gate packs and installs the three public packages in a clean external project. Publication and production deployment remain separate authorized actions.

Receiz proof memory follows one durable rule: first admission only, then append forever.

This standalone verifier advances package metadata, public entrypoints, service-worker identity, doctrine, and release records together through `pnpm release:lock 107.0.0`. That local qualification confirms release-surface continuity only; it does not publish packages, deploy production, mutate databases, push Git refs, or promote registry, SDK, MCP, AI-skill, emulator, generator, migration, cache, or release evidence into proof authority.
