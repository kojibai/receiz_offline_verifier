# Receiz v45.0.0

Release date: March 28, 2026

## Overview

`v45.0.0` is the production consolidation release for everything shipped since `v40.0.0`. This release rolls the platform forward from early Public Twin foundations into a production-ready World stack with photoreal live stages, stronger voice and video transport behavior, owned-runtime rendering, approval-gated twin promotion, deterministic proof bundles, and hardened calendar and booking flows.

The core rule does not change: the file remains the authority. Record, seal, share, and verify are still deterministic. Public `/v` routes, online viewers, profile delivery, and world surfaces provide context, portability, and operational reach around the same underlying artifact truth.

## Highlights

- Shipped the full Public Twin chat-stage architecture with separate dock and live-stage rendering paths, persistent session state, viewport-safe panel placement, stronger bootstrap reliability, and rendered-portrait support for the white-panel chat experience.
- Upgraded the photoreal shell with face-feature alignment, blink and breath envelopes, face-mapped eye and mouth motion, improved mouth articulation, and stronger photo-locked stage behavior.
- Hardened playback so baked video audio, separate audio assets, browser speech, and pulse fallback each use the right transport and lock behavior, with compact/mobile/master derivative selection and live-session video support.
- Expanded the owned and internal twin runtime stack with self-hosted TTS/render workers, speed and truth lanes, local model-pack discovery, latent bundle materialization, rerender quality gates, and production backend routing.
- Added full avatar lifecycle improvements: identity-still avatar generation, transparent asset handling, freshest-render preference, refresh propagation, preview locking, approval-gated photoreal promotion, and performance profile enrollment for voice and motion.
- Introduced deterministic trail and Twin Mind artifact flows, including signed bundle transport, PNG-embedded artifacts, verified zip download packaging, and public-safe receipt/bundle sanitization.
- Strengthened public profile intelligence and routing with better showcase selection, concierge streaming, business-lane controls, booking enforcement, welcome-flow preservation, and first-person conversation relay behavior.
- Expanded calendar infrastructure with sanitized event payloads, deterministic event bundles, stable event ordering, ledger-backed calendar support, booking answer reconciliation, and event bundle export routes.
- Added or refreshed major user-facing surfaces for `world`, `calendar`, `public-twin`, `payments`, `powered-by`, `developers/receiz-connect`, `faq`, `governance`, `legal`, `legal/compliance`, `security`, `standards`, `status`, `start-here`, and `trust`.
- Improved supporting platform pieces including media URL rewriting, profile media and storage routes, public wallet and ledger routes, chat user search, account and affiliate surfaces, and offline/PWA behavior.

## Detailed release notes

### Public Twin and World

- Delivered a more reliable World bootstrap, route hydration, session persistence, and profile/live-stage handoff model.
- Added public twin preview alignment, preview merge/lock behavior, deterministic visitor/bootstrap handling, and route continuity protections.
- Improved public policy enforcement for permissions, district access, embodiment mode, reply capability, and muted/generated/browser voice selection.
- Added world APIs for public profile, public stream replies, avatar generation, performance preview, performance profile, public view hydration, concierge stream, ventures, presence heartbeat, trail bundle download, Twin Mind import/export, and owner `world/me`.

### Voice, Playback, and Media

- Added browser speech fallback support across mobile, iPad-class, and desktop-class devices where allowed.
- Refined generated-voice fallback behavior so real media assets stay authoritative while browser speech remains available only when policy and transport allow it.
- Added compact shell delivery logic, mobile/master/preview mastered derivatives, raw-stream fallback, and live-session playback support.
- Tightened sync behavior for captions, transcript reveal, blink envelopes, Kai breath envelopes, and stage playback clocks.

### Photoreal Twin Runtime

- Added stronger internal and production worker contracts for cloned voice, render streaming, poster output, quality rerendering, and failure fallback behavior.
- Expanded the owned runtime to support local model packs, warmed lane caches, latent bundle materialization, split launchers, and production-quality photoreal settings.
- Added realism and approval gates so weak or drifted photoreal outputs are blocked before public promotion.

### Avatar and Identity

- Improved avatar source selection so the freshest valid render, canonical source, and profile media are chosen in the right order across stage, shell, and dock contexts.
- Added transparency-safe avatar processing and transparent-background generation guidance.
- Added avatar history/versioning support and refresh fanout across local browser transports.

### Calendar, Booking, and Proofs

- Added deterministic calendar event bundles with canonical hashing and visitor identity binding.
- Improved booking question enforcement and answer reconciliation against configured templates.
- Added storage URL rewriting and sanitization across calendar and bundle surfaces.
- Added ledger-backed trail receipts and downloadable verified trail bundle packages.

### Docs, Trust, and Platform Surfaces

- Refreshed major product and trust pages plus OG/Twitter assets for developer, legal, governance, security, standards, status, trust, payments, public twin, world, and onboarding surfaces.
- Added offline record-seal support and continued service worker/PWA updates.
