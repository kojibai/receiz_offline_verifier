## Title

Receiz `v85.0.0` - Official Proof-Native Sports, Market, Passport, and Speed Release

## Summary

`v85.0.0` is the official release that closes the full arc from `v80.0.0` to the current proof-native product:

```txt
Signal cards became provenance objects.
Sports became a live proof game economy.
Main Event became a settlement primitive with deterministic first paint.
Market and wallet returned to immediate deterministic ownership and settlement projection.
Series Passport became a proof object that gates the annual skill final.
Affiliate sharing uses the implemented referral system instead of a duplicate authority.
Release law, service-worker identity, public version labels, and product truth move together.
```

The pre-lock committed arc from `v80.0.0` to `2219e44061` contains 496 changed files, 172,799 insertions, and 3,947 deletions. The `v85.0.0` release-lock worktree adds the final release identity, docs, and the Series Passport strip label correction so the Arena strip shows Series-specific action truth: `$25 Passport` and `Every lineup counts`, not leaked Main Event prize-pool or cryptic seal language.

This release is not a generic feature bundle. It is the point where Receiz behaves like the implemented system already says it is: a proof-native artifact system with first-paint truth, deterministic local ownership, instant settlement projection, and global sync behind stronger known truth.

## Baseline From v80.0.0

`v80.0.0` left Receiz with the Signal Run proof/card foundation and the initial post-v80 card sharing and discovery work. The baseline already had proof-bearing game objects, public card surfaces, wallet/account surfaces, and route conventions, but the product arc after v80 made those primitives faster, more durable, more inspectable, and much more complete.

The release series after v80 did four major things:

- It made Signal Card provenance and history durable enough to inspect.
- It built Sports into a live MLB proof economy with owned cards, packs, scoring, tournaments, settlement, trade, vault, and conformance.
- It hardened first-paint law so proof, identity, ownership, settlement, market, wallet, and public witness surfaces reuse known truth immediately.
- It added the Series Passport as a proof object and public conversion surface tied into existing affiliate/referral mechanics.

## Product Statement

Receiz is a proof-native artifact system.

Files, identities, ownership states, settlement states, public witness surfaces, verification routes, Sports cards, Series Passports, wallet images, Cash Images, OTC upgrades, and market positions carry product truth.

Server, DB, session, cache, analytics, global sync, and external systems may hydrate, append, index, reconcile, and distribute. They do not outrank stronger local, deterministic, sealed, verified, or artifact truth.
